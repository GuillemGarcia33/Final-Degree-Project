# Genomes and Centromere Prediction Results

The genome FASTA files and the corresponding TRASH and quarTeT prediction results for the three species studied in this project are not stored directly in this repository due to file size limitations.

##  Available in the GitHub Release

You can download all datasets from the following release:

[Genome Data, Predictions and Results](https://github.com/GuillemGarcia33/Final-Degree-Project/releases)

## Contents per file

- `B_mori_results.zip`: Genome and prediction results for *Bombyx mori*
- `C_elegans_results.zip`: Genome and predictions for *Caenorhabditis elegans*
- `Rhynchospora_pubera_results.zip`: Genome and centromere predictions for *Rhynchospora pubera*
